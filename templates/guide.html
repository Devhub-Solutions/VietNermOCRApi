📡 Hướng dẫn sử dụng API
VNCV OCR + VietNerm — nhận dạng văn bản và trích xuất thực thể từ ảnh giấy tờ Việt Nam
💡
API nhận ảnh giấy tờ (CCCD, Hộ chiếu, GPLX…), trả về raw text OCR và NER result đã được phân tích theo từng loại giấy tờ. Hỗ trợ real-time progress qua WebSocket.
OCR
Trích xuất văn bản từ ảnh bằng ONNX engine
NER
Nhận dạng thực thể theo doc_type bằng PhoBERT
WS
Progress real-time qua WebSocket

⚡ OCR Scan
Nhận dạng văn bản và trích xuất thực thể từ ảnh giấy tờ
POST /ocr/v1/scan Upload ảnh → OCR + NER ▾
Query Parameters
TênKiểuBắt buộcMô tả
doc_type string optional
Loại giấy tờ cần nhận dạng
Mặc định: cccd · Xem bảng DocType ở trên
Body — multipart/form-data
TênKiểuBắt buộcMô tả
file File required
Ảnh giấy tờ · JPG, PNG, PDF
Ví dụ — cURL
bash 
curl -X POST "http://localhost:8000/ocr/v1/scan?doc_type=cccd" \
  -F "file=@/path/to/cccd.jpg"
Ví dụ — Python requests
python 
import requests

url  = "http://localhost:8000/ocr/v1/scan"
params = {"doc_type": "cccd"}

with open("cccd.jpg", "rb") as f:
    res = requests.post(url, params=params, files={"file": f})

data = res.json()
print(data["raw_text"])
print(data["ner_result"])

javascript 
const form = new FormData();

POST /api/doctypes Thêm doctype mới ▾
Request Body — application/json
FieldKiểuMô tả
key string required
Mã định danh duy nhất, khớp với doc_type VietNerm
label string required
Tên hiển thị trên giao diện
aliases string optional
Các tên khác, phân cách bằng dấu phẩy
enabled boolean optional
Bật/tắt hiển thị trên scanner
Mặc định: true
bash 
curl -X POST "http://localhost:8000/api/doctypes" \
  -H "Content-Type: application/json" \
  -d '{"key":"giay_khai_sinh","label":"Giấy khai sinh","aliases":"birth","enabled":true}'
PATCH /api/doctypes/{id} Cập nhật doctype ▾
ℹ
Tất cả field đều optional — chỉ gửi field cần thay đổi.
bash

  -H "Content-Type: application/json" \
  -d '{"enabled": false}'

🔌 WebSocket
Nhận tiến trình scan real-time — kết nối trước khi gọi POST /ocr/v1/scan
💡
WebSocket broadcast tới tất cả client đang kết nối khi có scan xảy ra. Kết nối càng sớm càng tốt để không bỏ lỡ event.
WS ws://localhost:8000/ws/scan Real-time progress ▾
Kết nối — JavaScript
javascript 
const ws = new WebSocket("ws://localhost:8000/ws/scan");

ws.onopen = () => {
  console.log("✓ WebSocket connected");
  ws.send(JSON.stringify({ action: "ping" }));
};

ws.onmessage = (event) => {
  const msg = JSON.parse(event.data);

  switch (msg.type) {
    case "pong":
      console.log("Server alive");
      break;
    case "progress":
      console.log(`Bước ${msg.step}: ${msg.msg}`);
      break;
    case "done":
      console.log("Hoàn tất!", msg.ner_result);
      break;
    case "error":
      console.error("Lỗi:", msg.msg);
      break;
  }
};

ws.onclose = () => console.log("Disconnected");
ws.onerror = (err) => console.error("WS Error", err);
Kết nối — Python (websockets)
python 
import asyncio, json, websockets

async def listen():
    uri = "ws://localhost:8000/ws/scan"
    async with websockets.connect(uri) as ws:
        await ws.send(json.dumps({"action": "ping"}))
        async for message in ws:
            msg = json.loads(message)
            print(msg)

asyncio.run(listen())
